## 👁️ Sobre a Elara

Inspirado na obra cinematográfica *Her*, o **Project Her** não é apenas mais um chatbot. A **Elara** é uma assistente de IA personalizada, desenvolvida para testar os limites da interação entre humanos e agentes autónomos. 

O diferencial reside na implementação de um **núcleo de personalidade**: a Elara possui um sistema de princípios e um "ego" que evolui conforme as interações, fugindo das respostas genéricas e programadas das IAs convencionais.

---

## 🛠️ Destaques Técnicos (Engineering)

Este projeto é o meu principal laboratório de **Engenharia de Prompts** e **Integração de Sistemas Escaláveis**:

* **🧠 Evolving Ego System:** Desenvolvimento de uma arquitetura que permite à IA manter uma consistência de personalidade e "humor" baseada em princípios fundamentais pré-estabelecidos.
* **⚡ Gemini API Integration:** Utilização dos modelos mais recentes do Google para processamento de linguagem natural de alta performance.
* **🤖 Prompt Engineering Avançado:** Estruturação de contextos complexos para garantir que a IA não apenas execute tarefas, mas compreenda as nuances emocionais e técnicas do utilizador.
* **⚙️ Automação de Contexto:** Gestão eficiente de janelas de contexto para manter a "memória" de longo prazo da assistente.

## 🚀 Stack Tecnológica

* **Linguagem Principal:** Python 3.13

## 📈 Visão de Futuro

O Project Her faz parte da minha jornada na **FIAP**, onde exploro a união entre **Automação de Processos** e **IA Generativa**. O objetivo final é criar um ecossistema onde a Elara atue como uma camada de inteligência autónoma, capaz de gerir fluxos de trabalho complexos enquanto mantém uma interface humana e empática.

## 📖 Como Configurar

1.  Clone o repositório.
2.  Crie um ambiente virtual: `python -m venv venv`.
3.  Instale as dependências: `pip install -r requirements.txt`.
4.  Configure a sua `API_KEY` no ficheiro `.env`.
5.  Execute o núcleo: `python main.py`.
